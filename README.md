# Talk2Data App

## Overview
Talk2Data is an AI-powered chat application with a FastAPI backend, localized MCP services, and a Next.js frontend.

## Components
- **Frontend**: Next.js application
- **Backend**: FastAPI server
- **NL2SQL MCP**: Service for Natural Language to SQL
- **Agentic MCP**: Service for agentic tools

## Running Locally
See `run_local.sh` for instructions.
